🤖 Claude Desktop Setup

Step 1: Sign Up & Get Your API Key

Click any "Sign Up" button above. For free tier, verify your email and receive your API key instantly. For paid tiers, complete payment and receive your key immediately.

Step 2: Locate Config File

Find your Claude Desktop configuration file:

📁 Windows:
%APPDATA%\Claude\claude_desktop_config.json
📁 macOS:
~/Library/Application Support/Claude/claude_desktop_config.json
📁 Linux:
~/.config/Claude/claude_desktop_config.json

Step 3: Add Configuration

Open the config file and add:

{ "mcpServers": { "security-orchestra": { "url": "https://security-orchestra-orchestrator.onrender.com", "transport": "sse", "headers": { "x-api-key": "YOUR_API_KEY_HERE" } } } }
⚠️ Important:
Replace YOUR_API_KEY_HERE with the actual API key you received via email!

Step 4: Restart & Test

Restart Claude Desktop, then ask:

"Show me what Security Orchestra agents are available"

You should see all 54 agents! 🎉
